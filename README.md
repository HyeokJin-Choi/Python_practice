insert(i,x) --> x를 리스트의 i번 원소로 삽입한다.
append(x) --> 원소 x를 리스트의 맨 뒤에 추가한다.
pop(i) --> 리스트의 i번 원소를 삭제하면서 알려준다.
remove(x) --> 리스트에서 (처음으로 나타나는) x를 삭제한다.
